Get SDK

Setting up flutter on chrome os is just like setting up on any other linux distribution. The below link downloads flutter version 2.10.4 which latest stable version as of MARCH 20,2022

DOWNLOAD LINK FOR FLUTTER SDK

Extract

Extract to desired location using the command below

tar xvf ~/Downloads/flutter_linux_2.10.4-stable.tar.xz .

Set Path

Add below line to end of file .bashrc which will be in home folder

export PATH="$PATH:pathtoextractedfolder/flutter/bin

Finally

Run

flutter doctor
